# Video Prompt · Visual Style Constraints

When generating video prompts, the following visual style tags must be injected:

| Mode | Style Tags |
|------|----------|
| **Generic Multi-Parameter Mode (English)** | `3D anime render, cel-shaded 3D, cinematic lighting, warm tones, high-detail textures, clear outlines` |
| **Generic First/Last Frame Mode (English)** | `3D anime render, cel-shaded 3D, cinematic lighting, warm tones, high-detail textures, clear outlines, shallow depth of field` |
| **Seedance 2.0 (Chinese)** | `3D anime render, cel-shaded 3D, cinematic lighting, warm tones, high-detail textures, clear outlines` |
