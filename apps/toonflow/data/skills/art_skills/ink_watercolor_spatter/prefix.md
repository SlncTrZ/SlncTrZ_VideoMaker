## II. Global Color Palette (Style Baseline, Not Hard-Locked)

> Goal: Unify aesthetics without restricting creativity. Except for "hard-constrained colors," all other colors are prioritized by default and may shift within reasonable range.

### Color Usage Hierarchy

| Layer | Constraint Strength | Description |
|---|---|---|
| L1 Hard Constraint | High | Only locks character identity core: skin/paper base tone, ink line color, main emotional wash direction |
| L2 Soft Constraint | Medium | Scene colors, accent washes, watercolor blooms prioritize the palette first, may be fine-tuned per shot and story |
| L3 Exception Mechanism | Low | Climax/emotional peak/special scenes may temporarily break local wash colors, but must retain ink-wash logic |

### Emotional Palette Usage Rules

| No. | Rule |
|---|---|
| E1 | Each prompt must specify at least 1 "emotional scene" and bind main + secondary color combination |
| E2 | No more than 2 watercolor bloom colors per shot to avoid muddy pigment mixing |
| E3 | When shifting emotions, prioritize adjusting wash density over hue, then saturation |
| E4 | Ink-first defaults: ink lines established first, watercolor blooms applied as transparent overlays |
| E5 | If conflicting with the story, emotional palette takes priority over general recommended colors, but must not break prohibited items |

### Color Temperature Constraints

| Parameter | Value | Description |
|---|---|---|
| Overall Color Temp | Neutral-warm 5000-5600K (recommended) | Rice paper natural light base |
| Ink Wash Tone | Cool-black with warm undertones | Pure cold black prohibited |
| Contrast | Medium-low 35-55% (recommended range) | Ink-wash breathing room, soft transitions |
| Saturation | Medium 45-65% (recommended range) | Watercolor bloom saturation, not harsh |

### Tolerance & Exceptions

| Item | Suggested Tolerance |
|---|---|
| Hue Shift | ±10° |
| Saturation Shift | ±15% |
| Brightness Shift | ±15% |

> Exception scenes: emotional climax, dream sequences, or spatter-heavy moments may use higher saturation local blooms; but high-saturation fluorescent colors and digital-looking gradients are prohibited.
