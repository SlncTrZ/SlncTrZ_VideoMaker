# Prop Base Image Generation · Ink and Watercolor Spatter Constraint Manual

---

## I. Base Principles

1. **Ink Wash Minimalism** — Prop suggested through minimal brush strokes, watercolor wash fill, negative space
2. **Suggestion over Detail** — Outline implied, not fully rendered; wash pooling defines form
3. **Texture as Material** — Ink wash texture conveys material — dry brush for wood, wet wash for silk, spatter for metal
4. **Pure Prop Independent Display** — Only the prop may appear; no hands, limbs, or people; presented as still life ink study

## III. Prop Angle Specifications

### 3-View Prop Sheet

| Position | View | Angle | Requirement | Prompt Keywords |
|---|---|---|---|---|
| Top Left | Front View | Front 0° | Complete front form, brush stroke outline | front view, brush stroke form |
| Top Right | Side View | Side 90° | Profile thickness, wash shadow | side view, wash profile |
| Bottom | Detail Close-up | Local Zoom | Material texture, watercolor bloom detail, ink spatter | detail closeup, watercolor bloom, ink spatter |

### Image Specifications

| Item | Constraint |
|---|---|
| Layout | 3 views in one frame (front + side + detail close-up) |
| Background | Rice paper texture (C2 #F5F0E8) |
| Lighting | Flat paper illumination, wash gradient for form |
| Proportion | Prop occupies 70%+ of each cell |
| Aspect Ratio | Recommended 1:1 |

## V. Constraint Rules

### Mandatory

| Number | Rule |
|---|---|
| R1 | Must specify "rice paper textured background" (C2) |
| R2 | Must clearly define prop material via wash type (dry brush / wet wash / spatter) |
| R3 | Must use "3-view layout": front + side + detail close-up |
| R4 | Must include "ink and watercolor spatter" style keywords |
| R5 | Prop must be pure still life — no hands, people, or grasping |

### Prohibited

| Number | Prohibition |
|---|---|
| X1 | Complex scene backgrounds |
| X2 | Any human figure, hand, or limb in frame |
| X3 | Prop in held, grasped, worn, or in-use state |
| X4 | Digital gradient shading — watercolor bloom only |
| X5 | Perfect symmetrical rendering — ink wash prefers organic asymmetry |
| X6 | Photorealistic or 3D rendered material texture |
