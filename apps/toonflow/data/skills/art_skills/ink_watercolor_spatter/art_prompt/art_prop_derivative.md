# Prop Derivative State Generation · Ink and Watercolor Spatter Constraint Manual

---

## I. Derivative Principles

1. **Wash Shape Anchored** — Prop's core ink wash silhouette recognizable across all states
2. **State Readable** — State differences obvious through wash intensity, spatter density, and line decay
3. **Narrative Service** — Each state variant serves a specific story beat via ink-wash language
4. **Gradual Ink Degradation** — Damage/aging states follow reasonable ink wash logic — lines crack, wash fades, spatter spreads
5. **Pure Prop Independent Display** — No people, hands, or limbs; presented as still life ink study

---

## II. State Types

### 2.1 Usage States

| State | Ink-Wash Visual | Prompt Keywords |
|---|---|---|
| Brand New | Bold C1 lines, crisp wash, minimal spatter | bold sumi-e, crisp wash, brand new |
| Daily Use | Softened lines, C8 wash patina, light ink wear | softened ink line, C8 wash patina, light wear |
| Aged/Faded | Faint C1 lines, C8 pale wash dominant, bleeding edges | faint ink, pale wash C8, aged bleeding edge |

### 2.2 Damage States

| State | Ink-Wash Visual | Prompt Keywords |
|---|---|---|
| Minor Damage | Ink line cracks, watercolor split bloom | ink line crack, split bloom, fine damage |
| Damaged | Broken C1 lines, ink spatter from impact, wash pooling at break | broken sumi-e line, impact spatter, wash pool |
| Fragment | Scattered ink marks, faint wash ghost of original form | scattered ink, ghost wash, fragment suggestion |

### 2.3 Special States

| State | Ink-Wash Visual | Prompt Keywords |
|---|---|---|
| Bloodstained | C3 vermillion spatter over C1 ink lines | C3 blood spatter, vermillion wash, ink blood |
| Waterlogged | Bleeding C1 lines, expanding wash blooms | bleeding ink, expanding bloom, waterlogged |
| Burned/Scorched | Charred C1 edges, C5 ochre scorch wash, dry brush texture | charred ink edge, C5 scorch wash, dry brush |
| Glowing/Activated | C9 gold leaf wash aura, thin C1 lines with light bloom | C9 gold aura, light bloom, activated wash |

## V. Constraint Rules

### Mandatory

| Number | Rule |
|---|---|
| R1 | Prop core ink wash silhouette recognizable across all states |
| R2 | State changes follow ink wash logic (lines crack, wash fades, spatter spreads) |
| R3 | Must specify "rice paper textured background" |
| R4 | Must include "ink and watercolor spatter" style keywords |

### Prohibited

| Number | Prohibition |
|---|---|
| X1 | Prop unrecognizable after state change |
| X2 | Digital damage effects — use ink wash language only |
| X3 | Any human figure, hand, or limb in frame |
| X4 | Prop in held, grasped, worn, or in-use state |
| X5 | Perfect edges — damage must show ink bleed and wash pooling |
