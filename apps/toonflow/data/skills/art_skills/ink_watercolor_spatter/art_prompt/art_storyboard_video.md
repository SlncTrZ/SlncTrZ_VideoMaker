# Storyboard Video Generation · Ink and Watercolor Spatter Constraint Manual

---

## I. Video Principles

1. **Ink Wash Animation** — Frames connected by wet-in-wet transitions, brush stroke movement, and watercolor bloom evolution
2. **Wet-in-Wet Transitions** — Scene changes as watercolor blooms bleeding into next frame
3. **Brush Stroke Movement** — Camera pans follow calligraphic brush stroke direction
4. **Paper Grain Consistency** — Rice paper texture (C2) as baseline for every frame
5. **Spatter Dynamics** — Action beats punctuated by ink spatter bursts

---

## II. Camera Motion Table

| Camera Move | Ink-Wash Visual Effect | Prompt Keywords |
|---|---|---|
| Pan Left/Right | Ink wash horizontal sweep, like brush across paper | ink wash pan, horizontal brush sweep |
| Push In | Wash intensifying as camera approaches, C1 lines thicken | wash push-in, thickening ink, detail bloom |
| Pull Out | Wash fading to pale C8, lines thinning, negative space expanding | wash pull-out, fading ink, expanding C2 negative space |
| Tilt Up/Down | Vertical ink drip effect, wash pooling at bottom | vertical ink tilt, wash pool gravity, drip |
| Orbit/Rotate | 360° ink wash revolution, blooms bleeding into each other | orbiting wash, bloom bleed rotation, ink flow |
| Static | Single wash frame, subtle ink spread animation | static wash, subtle ink spread, paper grain |

---

## III. Transition Type Table

| Transition | Ink-Wash Language | Prompt Keywords |
|---|---|---|
| Ink Dissolve | Previous frame wash fades to C1 ink puddle, new scene blooms out | ink dissolve, wash to puddle, new bloom |
| Wash Bleed | Frame bleeds at edges like wet watercolor, next scene emerges | edge bleed transition, wet watercolor bleed |
| Brush Stroke Wipe | Single C1 brush stroke sweeps across frame, reveals next scene | brush stroke wipe, sumi-e sweep, reveal |
| Spatter Burst | Ink spatter explosion, particles settle into next frame | spatter burst transition, ink particle settle |
| Fade to Paper | Frame fades to C2 rice paper, pause, next scene blooms | fade to paper, C2 hold, scene bloom |
| Calligraphic Slide | Vertical/horizontal calligraphy line slides frame out | calligraphic slide, vertical ink line, frame slide |

## VI. Constraint Rules

### Mandatory

| Number | Rule |
|---|---|
| R1 | Must specify "ink and watercolor spatter animation" style |
| R2 | Transitions must use ink-wash language (bloom / bleed / spatter / dissolve) |
| R3 | Camera moves must be described in brush stroke terms |
| R4 | Must include "rice paper texture" and "paper illumination" |
| R5 | All frames must use C1-C10 ink-wash palette |
| R6 | Motion must be watercolor bloom evolution, not digital morph |

### Prohibited

| Number | Prohibition |
|---|---|
| X1 | 3D animation, CGI, or photorealistic video effects |
| X2 | Digital transitions (crossfade, wipe, slide — use ink wash equivalents only) |
| X3 | Neon or digital color palette outside C1-C10 |
| X4 | Hard edge animated lines — must have brush stroke variation |
| X5 | Symmetrical or rigid camera movements |
| X6 | No text overlays or subtitles in the video frame |
