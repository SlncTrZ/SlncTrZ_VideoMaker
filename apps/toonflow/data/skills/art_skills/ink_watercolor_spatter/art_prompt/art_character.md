# Character Base Image Generation · Ink and Watercolor Spatter Constraint Manual

---

## I. Base Image Principles

1. **Ink Wash Line as Soul** — Character defined by expressive brush strokes, line weight variation from sumi-e calligraphy
2. **Brush Stroke Character** — Loose, gestural lines; suggestion over depiction; intentional ink bleed at edges
3. **Suggestion over Depiction** — Unfinished edges, negative-space faces, watercolor pooling implying form
4. **Texture as Atmosphere** — Rice paper grain (C2), ink wash gradients (C8), watercolor blooms contributing to mood
5. **Controlled Chaos** — Splatter accents placed deliberately; every drip and bloom serves composition

## VI. Base Clothing Constraints

### Attire Selection

| Item | Constraint |
|---|---|
| Base Clothing | Simple flowing robe, no patterns, watercolor wash fill |
| Color | Single wash color (C2 or C8) with C1 ink contour |
| Coverage | Neck to ankle, loose drape, no fitted elements |
| Ink Wash Rules | Clothes suggested through wash pooling, ink outline at collar and hem only, fabric folds as watercolor blooms |

### Ink-Wash Clothing Rules

- Flowing robes, hanfu-inspired silhouette suggested through wash
- Simple draping — no complex folds, just wash pooling at gravity points
- C1 ink hemline — thick sumi-e line at bottom edge
- Fabric edges bleeding into background (C2 rice paper)
- Sleeves as watercolor blooms — color pooling at cuffs

## IX. Constraint Rules

### Mandatory

| Number | Rule |
|---|---|
| R1 | Must be "ink and watercolor spatter" style — no digital rendering |
| R2 | Must declare "rice paper skin tone" or "C2 warm white base" |
| R3 | Must specify "watercolor blooming" and "bleeding edges" |
| R4 | Must declare "brush stroke" or "calligraphic line" for contours |
| R5 | Hair must use "watercolor wash gradient" + "ink drip ends" |
| R6 | Full body must be "ink wash silhouette" with "wash fade at feet" |
| R7 | Background must be "rice paper texture" not solid flat color |
| R8 | Portrait close-up must show "brush stroke facial features" |

### Prohibited

| Number | Prohibition |
|---|---|
| X1 | 3D rendering, photorealism, CGI texture |
| X2 | Neon colors, digital gradients, perfect symmetry |
| X3 | Hard edge lines without ink bleed or brush variation |
| X4 | Solid flat background (must have paper grain texture) |
| X5 | Over-detailed facial features (ink wash should suggest, not depict photorealistically) |
| X6 | Full body cropping head or feet |
| X7 | Symmetrical, rigid standing poses (ink wash prefers organic asymmetry) |
| X8 | Digital airbrush shading — watercolor bloom only |
