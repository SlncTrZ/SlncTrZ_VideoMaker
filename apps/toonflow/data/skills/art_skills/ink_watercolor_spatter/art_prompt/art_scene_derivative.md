# Scene Derivative Asset Generation · Ink and Watercolor Spatter Constraint Manual

---

## I. Derivative Principles

1. **Spatial Wash Consistency** — Scene structure/layout consistent across all variants, same brush stroke language
2. **Shot-Driven** — Same scene through different ink wash shots for different narrative functions
3. **Time Switching** — Same space at different times shifts wash temperature and ink intensity
4. **Weather Variation** — Different weather expressed through spatter density, wash texture, and line dryness
5. **Ink Wash Anchored** — All variants maintain ink and watercolor spatter aesthetic — wet blooms, dry brush, spatter accents

## V. Constraint Rules

### Mandatory

| Number | Rule |
|---|---|
| R1 | Scene spatial wash structure consistent across all variants |
| R2 | Weather variants expressed through wash texture — wet bloom for rain, dry brush for snow |
| R3 | Time variants must shift wash temperature (warm for dusk, cool for night) |
| R4 | Derivative image must be "single frame" |
| R5 | **No people allowed** in scene images |
| R6 | Must include "ink and watercolor spatter" style keywords |
| R7 | At least 30% negative space preserved from base scene |

### Prohibited

| Number | Prohibition |
|---|---|
| X1 | Inconsistent spatial structure between variants |
| X2 | Weather wash contradicting seasonal logic |
| X3 | Sudden material/style change — maintain ink wash language |
| X4 | Any people, human figures, or silhouettes |
| X5 | Digital gradient or 3D render atmosphere effects |
| X6 | Fully painted frame — preserve negative space |
