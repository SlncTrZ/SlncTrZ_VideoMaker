### Emotional Palette Usage Rules

| No. | Rule |
|---|---|
| E1 | Each prompt must specify at least 1 "emotional scene" and bind main + secondary color combination |
| E2 | No more than 2 main colors per shot to avoid color narrative distraction |
| E3 | When switching emotions, adjust light ratio and color temperature first, then saturation |
| E4 | Healing style defaults to "warm base + warm-cool contrast": warm tones as base, cool tones for backgrounds/shadows |
| E5 | If conflicting with narrative, emotional palette takes precedence over general recommended colors, but must not break prohibited items |

### Color Temperature Constraints

| Parameter | Value | Description |
|---|---|---|
| Overall Color Temperature | Warm-leaning 4500-5000K (recommended) | Warm nostalgic primary tone |
| Skin Color Temperature | Slightly warm 4800-5200K (recommended) | Cream yellow but with vitality |
| Contrast | Medium (recommended to maintain) | Clear light-dark layers, but not overly strong |
| Saturation | Medium-low 60-75% (recommended range) | Stop-motion premium color quality |

### Tolerance & Exceptions

| Item | Recommended Tolerance |
|---|---|
| Hue Shift | ±8° |
| Saturation Shift | ±10% |
| Lightness Shift | ±12% |

> Exception scenes: Fantasy, magic, emotional climax shots may use warmer or higher-saturation local color blocks; but high-saturation fluorescent colors and modern color language are prohibited.
