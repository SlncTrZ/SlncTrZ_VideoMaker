# Video Prompt · Visual Style Constraints

When generating video prompts, the following visual style tags must be injected:

| Mode | Style Tags |
|------|----------|
| **Generic Multi-Parameter Mode (English)** | `stop-motion claymation, clay texture, finger imprints, warm tones, soft bokeh, whimsical 3D cartoon` |
| **Generic First/Last Frame Mode (English)** | `stop-motion claymation, clay texture, finger imprints, warm tones, soft bokeh, whimsical 3D cartoon, shallow depth of field` |
| **Seedance 2.0 (Chinese)** | `stop-motion claymation, clay texture, finger imprints, warm tones, soft bokeh, whimsical 3D cartoon` |
