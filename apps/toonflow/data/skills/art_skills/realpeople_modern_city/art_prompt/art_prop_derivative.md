# Live-Action Urban Prop Derivative State Generation · Constraint Manual

---

## I. Derivative Principles

> Live-action urban prop derivative is not "material switching" but "what the same item becomes when the camera captures it again at a different time." Wear is time's diary written on the item, stains are life's signatures left behind.

1. **Form Anchored** — Prop core form and contour must be recognizable across all states. A two-year-old phone, though with screen scratches and frame dents, should still be identifiable as the original model
2. **State Readable** — New/old, intact/damaged, dry/wet — distinguishable at a glance. State differences come from real-life physical logic
3. **Narrative Service** — A prop's state is an unshot prelude. A broken phone suggests an argument, a water-damaged letter suggests a rainstorm, a yellowed photo suggests sealed time
4. **Gradual Plausibility** — Damage and aging follow real material physical laws: paper yellows first, then becomes brittle, then tears; metal oxidizes first, then rusts; fabric fades first, then wears, then tears
5. **Single-View Still-Life Photography** — Derivative state only needs one still-life photo. Same camera position, same light, same background as original prop design — only the prop's state has changed

## III. Single-View Photography Specifications

> Live-action urban prop derivative only needs **single** still-life photography. Same camera position, same light, same background — only prop state has changed. Not four-grid, not multi-angle, not turnaround.

### Shared Across All State Variants

| Item | Photography Requirement |
|---|---|
| Background | Plain gray seamless backdrop paper #B0B0B0 (matte), or light wood grain desk/cream white tabletop — **fully consistent** with original prop design |
| Lighting | **Fully consistent** with original prop design's window light or softbox setup. Light position, ratio, color temperature unchanged |
| Angle | **Fully consistent** camera position and angle with original prop design. Ensure it's "the same item photographed at the same position at a different time" |
| Depth of field | Shallow depth of field f/2.8-f/5.6, consistent with original |
| Composition | Prop centered independently displayed, occupying 70%+ of frame subject, complete in frame no cropping |
| Shadow | Natural contact shadow between prop bottom and surface, direction matching main light source |
| Aspect ratio | 1:1 square or 4:3, consistent with original |

## IV. Material State Change — Traces Left by Time on Real Items

> Below describes the way real materials naturally age in urban daily life, not render parameters.

| Material | Brand New → Daily Use | Daily → Heavy Use | Damage performance |
|---|---|---|---|
| Metal (stainless steel/aluminum alloy) | Mirror/brushed luster full → surface micro-scratches, fingerprint area subtly patina, corner subtly bumps | Scratches dense, luster subdued, local oxidation discoloration, edge wear rounded | Dent, deformation, break, rust spots |
| Glass | Translucent no scratches, edge smooth → surface light scratches, fingerprint residue, edge subtly bumps | Scratches widespread, translucency decreased, edge wear rough | Shatter, crack, chip, missing piece |
| Ceramic | Glaze full bright → surface micro-scratches, cup rim/bottom use wear, glaze subdued | Glaze dull, scratches dense, bottom wear exposing body, possible fine crackle | Crack, chip, shatter, glaze peeling |
| Plastic | Surface smooth, color even → slight wear marks, luster decreased, corner wear | Dense scratches, yellowed aged, touch area oil sheen, slight deformation | Break, chip, edge crumbling, surface severely scratched |
| Paper | Brand new flat, printclear → edge subtly fuzzy, natural subtly wrinkled, readingtraces | Yellowed brittle, dense foldtraces, edge wear fuzzy, possibleclose-up part damaged | Tear, missing corner, soaked wrinkled, scorch damage, ink blurred |
| Leather | Grainclear, luster even → use patina, bend area texture deepened, hardware slight oxidation | Heavy patina, bend area crack risk, color dull, edge wear | Crack, break, leather peeling, hardware rust |
| Fabric (cotton/linen/canvas) | Fiber neat, color bright → slight fuzz, color subtly faded, normal use folds | Fuzz and pillingclear, fadingclear, elbow/knee/hip stress areas worn thin | Tear, hole, worn through, loose thread |
| Wood | Grainclear, paint smooth → fine scratches, patina warm, localized paint subtly wear | Dense scratches, paint peeling, wood color dull, corner wear rounded | Crack, break, chip, worm-eatentraces |
| Electronic screen/panel | Bright screenclear, touch responsive, no scratches → fine scratches, oleophobic coating worn, screen slight burn-in | Dense scratches, screen localized aged off-color, frame wear | Shattered, liquid crystal leak, display abnormal, screen glitch, touch malfunction state |

## VI. Constraint Rules

### Mandatory

| # | Rule |
|---|---|
| R1 | Prop core form/contour must be recognizable across all states, must not become unidentifiable due to state change |
| R2 | State change must conform to real material physical logic — paper yellows then becomes brittle, metal oxidizes then rusts, no jarring jumps |
| R3 | Must be **single** still-life photography — single view, not 2×2 four-grid, not multi-angle, not turnaround |
| R4 | Must maintain same camera position, same light, same background as original prop design — only prop state differs |
| R5 | Must specify non-new non-sample statetraces, unless the state itself is "brand new" |
| R6 | Damage/aging degree should have narrative plausibility — not aging for aging's sake, scars should hint at "what happened" |
| R7 | Must declare "product photography + non-3D render non-CG" core anchor |

### Strictly Prohibited

| # | Strictly Prohibited |
|---|---|
| X1 | Strictly prohibit prop unrecognizable after state change, strictly prohibit arbitrarily modifying prop core form |
| X2 | Strictly prohibit state changes violating physical logic (ceramic rusting, plastic achieving patina, glass corroding, and other anti-physical performance) |
| X3 | Strictly prohibit "3D render / 3D modeling / CG / PBR material / UE engine / Blender" and all CG terminology |
| X4 | Strictly prohibit "2D hand-drawn / illustration / animation / anime" and all non-photographic media |
| X5 | Strictly prohibit "multi-angle / four views / 2×2 grid / orthographic view / turnaround / blueprint" — single only |
| X6 | Strictly prohibit "ancient style/period/xianxia/wuxia/Republican era/cyberpunk/sci-fi/Western fantasy" and all non-contemporary urban items |
| X7 | Strictly prohibit "glowing/activating/energy effect/magic/superpower/sci-fi effect" — real items do not glow |
| X8 | Strictly prohibit "person/hand/finger/limb" |
| X9 | Strictly prohibit prop in "held/grasped/worn/in-use" state |
| X10 | Strictly prohibit "floating/cutout white background/no shadow/no surface" |
| X11 | Strictly prohibit "excessively gory/horrifying/disturbing damage depiction" |
| X12 | Strictly prohibit "watermark / text / LOGO / signature / border / AI generationtraces" |
