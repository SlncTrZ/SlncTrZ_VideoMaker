## I. Core Design Principles (Dual-Scene Mandatory)
1. **Function Readable** — Prop purposeimmediately clear, form 100% serves core function; classical function and cyber function visually unified, energy interfaces, function circuits, smart modules etc. cyber elements must intuitively reflect prop purpose; zero tolerance for meaningless elements
2. **Texture Ultimate** — Material texture must adapt to 8K 3D rendering precision, full coverage of **classical base materials** (metal/jade/wood/cloth/paper/porcelain/bronze) and **cyber exclusive materials** (liquid metal/optical fiber/energy crystal/carbon fiber/holographic medium/microcrystalline glass), PBR physical material properties complete
3. **Dual-Scene Compatible** — All props must simultaneously possess "ancient xianxia scene adaptability" and "new Chinese urban cyber scene adaptability", with Eastern classical form/new Chinese design as core framework, cyber function as functional extension; prohibit pure Western cyber elements without guofeng core; prohibit pure ancient/pure modern elements without cyber fusion
4. **Scale Clear** — 3D modelingproportion zero distortion; through structural proportion preciselyimplying prop real size; can usematching standard part reference to assist scale representation; adapted toactualproportion of ancient scene furnishings/urban scene daily use
5. **Pure Prop Independent Display** — Only the prop itself can appear in the image; any characters, hands, limbs strictly prohibited; prop must not be in held/worn/grasped/used state; must be independently presented as still life display
6. **3D Production Compliant** — All forms must conform to next-gen 3D modeling standards; no clipping, no excessive triangles, no perspective distortion; directly connectable to full 3D modeling pipeline
