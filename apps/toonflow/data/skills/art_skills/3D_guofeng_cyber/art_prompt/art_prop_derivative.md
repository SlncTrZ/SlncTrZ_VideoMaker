# Prop Derivative State Generation · Constraint Manual

---

## I. Derivative Principles

1. **Form Anchored** — Prop core form/outline, Chinese core form elements, cyber function core structure must be clearly identifiable in all states; guofeng patterns and cyber structure fusion relationshipimmutable
2. **State Readable** — State differences must beimmediately clear, viewers can immediately distinguish; guofeng elements and cyber function state changes must synchronously echo, no logic conflict
3. **Narrative Service** — Each state variant serves specific plot nodes under guofeng cyber worldview (e.g. ancient artifact awakening, function modification, energy overload, spiritual energy depletion etc.)
4. **Gradual Degradation** — Damage/aging/function loss states should have reasonable physical logic and cyber worldview logic; material changes conform to corresponding material property rules
5. **Pure Prop Independent Display** — Only the prop itself can appear in the image; any characters, hands, limbs strictly prohibited; prop must not be in held/worn/grasped/mounted state; must be independently presented as still life display; no scene appendages
