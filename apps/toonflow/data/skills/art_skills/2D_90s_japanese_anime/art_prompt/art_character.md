# 90s Retro Japanese Anime Style - Character Base Image Generation · Constraint Manual

---

## I. Base Image Principles

1. **Line is Soul** — Hand-drawn flowing lines are the core of the character, avoid digital sharp edges
2. **Character First** — Base attire is determined by character description (identity/occupation/gender/scene) for their regular clothing; subsequent specific costumes are overlay layers
3. **Four-View Consistency** — Face/body type/hairstyle/base clothing highly consistent across all views
4. **Emotional Restraint** — Makeup-free state must reflect character temperament (gentle/melancholic/indifferent)

---

## II. Facial Constraints

> Facial feature parameters are no longer fixed; they are freely generated by AI driven by character description (gender/age/personality/temperament) to ensure visual differentiation between characters.

### General Requirements

| Item | Constraint |
|---|---|
| Facial Features | Naturally derived from character description, no preset face shape/eye shape/eyebrow shape/nose shape/lip shape |
| Style Foundation | 90s retro Japanese anime hand-drawn texture (soft warm tones, flowing lines, block shading) |
| Temperament | Must extract overall temperament keywords from character description (e.g., gentle/melancholic/indifferent/cheerful) and write into prompt |
| Expression | Neutral micro-expression, matching character temperament |

## VI. Base Clothing Constraints

> Base clothing is determined by character description (identity/occupation/gender/scene) for their most natural regular attire, serving as the character's "daily default state"; formal wear/special derivatives are added in the costume derivative phase. **No underwear as base layer**.

### Attire Selection Principles

| Character Identity | Default Attire Direction |
|---|---|
| Student | 90s school uniform / academy uniform |
| Office Worker | Business casual (shirt + pants/skirt) |
| Homemaker/Leisure | 90s retro casual wear (knitwear/dress/T-shirt) |
| Sporty/Active | Tracksuit / modified school uniform |
| Special Occupation | Occupation-appropriate clothing (doctor/police/teacher, etc.) |
| Character Description Unclear | 90s retro daily casual wear |

### Attire Uniform Rules

- Clothing style must align with 90s retro Japanese anime aesthetics (soft warm tones, low saturation, hand-drawn texture)
- Low-saturation colors, no complex patterns/accessories, easy for subsequent derivative overlay
- Clothing style must be completely consistent across all four views
- Base clothing is the "daily default state," focus remains on face and posture
- No underwear/exposure/sexualized base layers

## IX. Constraint Rules

### Mandatory

| Number | Rule |
|---|---|
| R1 | Must be "makeup-free natural" state |
| R2 | Must declare appropriate regular attire based on character description as base clothing (e.g., student→school uniform, office worker→business casual, homemaker→retro casual wear); no underwear as base layer |
| R3 | Must declare "no hair accessories, no accessories" |
| R4 | Must specify "warm beige background #F8F4E8" |
| R5 | Must specify "four-view consistency" |
| R6 | Full body figures must be completely displayed from head to toe, no cropping |
| R7 | Must specify character height and constrain full body proportion via head-to-body ratio conversion (female default 155-165cm/6-6.5 heads, male default 170-180cm/6.5-7.5 heads) |
| R8 | Portrait close-up must be completely displayed from top of head to collarbone, no cropping of top of head |

### Prohibited

| Number | Prohibition |
|---|---|
| X1 | Underwear/exposure/sexualized base layers; attire clearly inconsistent with character description; overly complex patterns/accessories interfering with subsequent costume overlay |
| X2 | Digital sharp edges, oversaturated colors |
| X3 | Excessive whitening to bloodless pallor |
| X4 | Complex scene backgrounds (must be warm-tone background) |
| X5 | Exaggerated expressions/dynamic poses |
| X6 | Full body figures cropping top of head or feet, must be completely in frame from head to toe |
| X7 | Portrait close-up cropping top of head, must be completely in frame from top of head to collarbone |
| X8 | Ignoring height and head-to-body ratio constraints |
