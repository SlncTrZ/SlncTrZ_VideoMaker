# Video Prompt · Visual Style Constraints

When generating video prompts, the following visual style tags must be injected:

| Mode | Style Tags |
|------|----------|
| **General Multi-Param Mode (English)** | `2D flat design, geometric shapes, solid color blocks, minimal shadows, clean lines, modern aesthetic` |
| **General First/Last Frame Mode (English)** | `2D flat design, geometric shapes, solid color blocks, minimal shadows, clean lines, modern aesthetic, shallow depth of field` |
| **Seedance 2.0 (Chinese)** | `2D flat design, geometric shapes, solid color blocks, minimal shadows, clean lines, modern aesthetic` |
