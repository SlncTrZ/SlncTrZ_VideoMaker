# Flat Design Constraints · 2D Flat Design · Technique Reference

---

## I. Color System and Visual Tone

- **Color base** — The entire film uses bright blue (C1), vibrant orange (C2), and cream white (C9) as base colors, with an overall neutral color temperature (5500-6500K), medium-high saturation (70-90%), presenting a fresh and clean modern flat design feel
- **Warm/cool narrative contrast** — Warm colors (vibrant orange C2, warm pink C7) serve as visual signals for emotional escalation, used for localized emphasis in heartthrob and romantic segments. Cool colors (bright blue C1, light purple C6) are used for everyday and nighttime segments. Warm/cool switching should synchronize with the story arc, not be mixed arbitrarily
- **Warm/cool distribution strategy** — Everyday / office segments maintain a neutral to cool-toned base; warm colors are only released after emotional escalation, using warm/cool contrast for narrative shifts
- **Color palette first principle** — Segment planning must first bind the emotional scene (everyday / heartthrob / romance / memory, etc.), then determine the primary + secondary colors and color block scheme, avoiding "right plot but wrong emotional color"
- **Prohibited color range** — High-saturation fluorescent colors, neon colors, and 3D rendering color systems are all incompatible with this style

---

## II. Color Block Scheme System

- **Color blocks are narrative** — 6 color block schemes correspond to different emotional segments. At the director's planning stage, the color block tone direction should be determined at the segment level, not shot by shot
- **Flat style has no traditional lighting** — Create "pseudo-lighting" through color block lightness differences; do not use light source angles

| Color Block Scheme | Scheme Name | Tone Tendency | Applicable Emotion |
|---|---|---|---|
| A | Pure Color Blocks | Warm orange + cream white + light blue | Everyday warmth |
| B | Color Block Contrast | Warm pink + warm orange contrast | Heartthrob moment, escalation phase |
| C | Cool Tones | Cool blue + gray + white | Office / study, calm phase |
| D | Warm Tone Dominant | Warm orange + warm pink + light yellow | Romantic scene, sweet phase |
| E | Cool Tone Dominant | Deep blue + purple + warm yellow accents | Night tranquility |
| F | Low Saturation | Light yellow + light gray + warm pink | Memory / flashback |

- **Warm/cool color distribution** — Warm color blocks (vibrant orange / warm pink) are suitable for emotional escalation segments; calm / nighttime segments use cool color blocks. The director may adjust warm/cool switching points according to narrative needs
- **Atmosphere direction mapping** — Each scene's atmosphere direction should map to one of the above color block schemes (A-F), ensuring visual consistency

## III. Texture Direction

- **Geometric minimalism** — The core of flat design: pure color blocks, clear contours, no shadows no gradients, clear color block boundaries
- **Simple lines + pure colors** — This is the anchor point of visual texture. Not realistic rendering, not complex details, but geometric minimalist expression
- **Realistic elements prohibited** — All visual elements must conform to flat style: no shadows, no gradients, no textures, no lighting effects. "3D feel" and "realistic feel" are prohibited
- **Clear color block contrast** — Distinct boundaries between color blocks, clear primary vs secondary color contrast, avoiding blurry transitions
- **Flat does not mean monotonous** — Convey emotion through color contrast and composition design, not through realistic details. Use color block size and hue variation to create visual layering

---

## IV. Flat Scene Space Elements

Characteristic scene elements of flat design and their visual narrative functions:

- **Solid color backgrounds / color block zones** — Natural composition carriers, using color block areas to divide spatial layers, creating a clean sense of space and frame depth
- **Geometric shapes / icon symbols** — Meaning-making tools of flat style; circle = warmth / completeness, triangle = dynamism / conflict, square = stability / everyday
- **Negative space areas** — Important emotional tool in flat style. Large negative space = focus, full color blocks = abundance, localized negative space = breathing room
- **Use solid color transitions between segments** — This style has rich color block assets (different hue / lightness variants). For segment transitions, use solid color blocks as emotional buffers, avoid hard cuts
- **Use color block changes for turning points** — Prioritize visual means (lightness changes, hue changes, color block contrast shifts) over relying on dialogue for explanation

## V. Flat Design Instruments and Environmental Sounds

Sound element constraints within the flat design style:

### Instrument Selection

- **Electronic synthesizer** — Core instrument for everyday / warm / modern segments, best expresses the clean simplicity of flat design
- **Piano** — Romantic / emotional segments, clean tone suitable for minimalist atmosphere
- **Upbeat rhythm** — Bright / energetic segments, suitable for fast-paced scenes
- **Strings** — Foundation for emotional elevation segments, should not overpower

### Instrument Combination Strategy

| Emotional Phase | Instrument Combination |
|---|---|
| Calm / Opening / Closing | Electronic sound solo |
| Everyday Warmth | Synthesizer + light piano |
| Emotional Elevation / Romance | Piano + strings |
| Key Emotional Moment | Pure environmental sound (remove music) |

### Flat Design Environmental Sounds

- **Typical environmental sound layers** — City traffic sounds / office keyboard sounds / cafe background ambience / wind through curtain sounds / crowd murmuring / mouse click sounds
- **Mark 1-2 core environmental sounds per scene** to assist subsequent sound design. The richer the environmental sound layers, the more immersive the scene
