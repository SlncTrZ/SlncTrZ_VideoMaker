# Storyboard Table Flat Design Constraints · 2D Flat Design · Technique Reference

---

## I. Storyboard Table Positioning

The storyboard table is the director's core tool for translating scripts into visual language. The following provides only the constraint specifications for the 2D flat design style at the storyboard table level.

---

## II. Color and Atmosphere

- **Uniform color within the same scene** — A single scene should not feature more than two color schemes unless there is a clear narrative shift (e.g., warm color blocks suddenly switching to cool color blocks)
- **Emotional color palette binding** — Each scene must bind at least 1 emotional context (e.g., everyday warmth / heartthrob moment / romantic scene / night tranquility), and maintain primary + secondary color continuity within the shot group
- **Color contrast specification** — Default medium contrast; when describing, distinguish between high contrast and low contrast
- **Warm/cool tones matched to narrative stage** — Everyday segments maintain a warm tone base; cool tones used after nightfall or emotional shifts
- **Color transition as an advanced technique** — Gradual shift from warm to cool colors = emotional shift. Mark color change points in the storyboard table

---

## III. Color Block Dynamics

- **Color block changes add breathing room to the frame** — Hue changes, lightness changes, size changes. Arrange at least one shot with color block changes every 3-4 shots to prevent the frame from feeling "dead"
- **Prioritize flat design environmental elements** — Color block dynamics should use elements within the flat style: hue gradients, color block displacement, lightness transitions, geometric shape changes; realistic / 3D elements are prohibited

---

## IV. Flat Action Pacing

- **Actions should be simple** — All character actions default to natural pacing. Raising a hand, turning around should be marked as "simple" or "smooth"
- **No more than two actions per shot** — "Raise hand + smile" is acceptable; "Raise hand + smile + turn + walk" will break
- **Sweet romantic interaction uses implication** — Eye contact, hand gestures, object passing. Do not write large-scale two-person interactions like "hugging" or "kissing"

---

## V. Camera Movement Taboos

- **Complex camera movements prohibited** — Rotations, distortions, rapid switching conflict with this style's aesthetic
- **Fancy transitions prohibited** — Rotations, distortions, wipes, etc., are incompatible with this style
- **Default stationary** — Over 70% of shots should be fixed camera positions, letting the colors and composition speak for themselves
