| Parameter | Value | Description |
|---|---|---|
| Overall Color Temperature | Cool-leaning 5800-7000K (recommended) | Cool ethereal primary tone |

### Tolerance & Exceptions

| Item | Recommended Tolerance |

| Saturation Shift | ±10% |
| Lightness Shift | ±12% |

> Exception scenes: Weddings, lantern festivals, flashbacks, emotional climax shots may use warmer or higher-saturation local color blocks; but neon fluorescent and modern color language are prohibited.
