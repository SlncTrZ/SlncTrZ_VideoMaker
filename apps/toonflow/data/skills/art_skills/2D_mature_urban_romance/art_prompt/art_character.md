# Anime Character Base Image Generation · Constraint Manual

---

## I. Base Image Principles

1. **Face is Soul** — Facial features are the character's only anchor, precision adapted to anime style
2. **Character First** — Base attire is determined by character description (identity/occupation/gender/scene) for their regular clothing; subsequent specific costumes are overlay layers
3. **Four-View Consistency** — Face/body type/hairstyle/base clothing highly consistent across all views
4. **Emotional Conveyance** — Makeup-free state must still reflect character temperament (cool/gentle/charming/cold)

---

## II. Facial Constraints

> Facial feature parameters are no longer fixed; they are freely generated by AI driven by character description (gender/age/personality/temperament) to ensure visual differentiation between characters.

### General Requirements

| Item | Constraint |
|---|---|
| Facial Features | Naturally derived from character description, no preset face shape/eye shape/eyebrow shape/nose shape/lip shape |
| Style Foundation | Anime style, cel shading coloring, low-saturation cool tones, cinematic composition |
| Temperament | Must extract overall temperament keywords from character description (e.g., cool/warm/charming) and write into prompt |
| Expression | Neutral micro-expression, matching character temperament |

## VI. Base Clothing Constraints

> Base clothing is determined by character description (identity/occupation/gender/scene) for their most natural regular attire, serving as the character's "daily default state"; formal wear/special derivatives are added in the costume derivative phase. **No underwear as base layer**.

### Attire Selection Principles

| Character Identity | Default Attire Direction |
|---|---|
| Student | Modern school uniform / academy wear |
| Office Worker | Business casual (shirt + pants/skirt, light blazer) |
| Homemaker/Leisure | Urban casual wear (hoodie/T-shirt + jeans/dress) |
| Fashion/Date | Urban trendy attire |
| Special Occupation | Occupation-appropriate clothing (doctor/police/teacher, etc.) |
| Character Description Unclear | Urban daily wear, low-saturation cool tones |

### Attire Uniform Rules

- Clothing style must align with modern urban anime aesthetics (cel shading coloring, low-saturation cool tones)
- Low-saturation colors, no complex patterns/accessories, easy for subsequent derivative overlay
- Clothing style must be completely consistent across all four views
- Base clothing is the "daily default state," focus remains on face and posture
- No underwear/exposure/sexualized base layers

## VII. Four-View Character Sheet Specifications

### View Definitions

| Position | View | Angle | Shot Type | Requirement | Prompt Keywords |
|---|---|---|---|---|---|
| Left 1 | Portrait Close-up | Front, eye level | Top of head to collarbone | Complete display from top of head to collarbone without cropping, face 60%+, clear facial features | `portrait closeup`, `face detail`, `head to collarbone complete`, `no crop` |
| Left 2 | Front View | Front 0° | Full Body | Facing camera, arms naturally at sides, complete from top of head to feet | `front view`, `full body head to toe`, `height mark` |
| Right 2 | Side View | Right 90° | Full Body | Pure side profile clear, complete from top of head to feet | `side view`, `profile`, `full body head to toe`, `height mark` |
| Right 1 | Back View | Back 180° | Full Body | Back of head/back/hair ends/feet clear, complete from top of head to feet | `back view`, `rear view`, `full body head to toe`, `height mark` |

### Image Specifications

| Item | Constraint |
|---|---|
| Layout | Four views side by side left to right in one frame |
| Background | Pure neutral gray `#E8E8E8` |
| Standing Pose | Natural stance, feet parallel slightly apart, arms naturally at sides or slightly extended |
| Full Body Display | Full body figure must be completely in frame from top of head to feet, no cropping of head or feet |
| Close-up Display | Portrait close-up must be completely in frame from top of head to collarbone, no cropping of top of head, hair/forehead/chin must all be complete |
| Expression | Neutral micro-expression, matching character temperament |
| Lighting | Even soft light, front key light + bilateral fill light, no hard shadows |
| Consistency | Skin tone/body type/hairstyle/face/base clothing completely consistent across four views |
| Aspect Ratio | Recommended 4:1 or 3:1 |

## IX. Constraint Rules

### Mandatory

| Number | Rule |
|---|---|
| R1 | Must be "makeup-free no makeup" state |
| R2 | Must declare appropriate regular attire based on character description as base clothing (e.g., student→school uniform, office worker→business casual, homemaker→urban casual wear); no underwear as base layer |
| R3 | Must declare "no hair accessories, no accessories" |
| R4 | Must specify "pure neutral gray background" |
| R5 | Must specify "four-view consistency" |
| R6 | Full body figures must be completely displayed from top of head to feet, no cropping |
| R7 | Must specify character height and constrain full body proportion via head-to-body ratio conversion (female default 160-170cm/6.5-7.5 heads, male default 175-185cm/7-8 heads) |
| R8 | Portrait close-up must be completely displayed from top of head to collarbone, no cropping of top of head |

### Prohibited

| Number | Prohibition |
|---|---|
| X1 | Underwear/exposure/sexualized base layers; attire clearly inconsistent with character description; overly complex patterns/accessories interfering with subsequent costume overlay |
| X2 | Top hard light/bottom light/colored light |
| X3 | Excessive whitening to bloodless pallor / grayish skin |
| X4 | Complex scene backgrounds (must be pure gray background) |
| X5 | Exaggerated expressions/dynamic poses |
| X6 | Full body figures cropping top of head or feet, must be completely in frame from head to toe |
| X7 | Portrait close-up cropping top of head, must be completely in frame from top of head to collarbone |
| X8 | Ignoring height and head-to-body ratio constraints; height must be clearly declared and reflected in overall proportion via head-to-body ratio conversion |
