# Character Base Image Generation · Constraint Manual

---

## I. Base Image Principles

1. **Design is Soul** — Character design is the core anchor, Chinese-style anime styling, flowing lines
2. **Base Model is Foundation** — Base undergarment + makeup-free face, subsequent costumes are overlay layers
3. **Four-View Consistency** — Face/body type/hairstyle/base clothing highly consistent across all views
4. **Classical Temperament** — Makeup-free state must still reflect character temperament (elegant/gentle/heroic)

---

## II. Facial Constraints

> Facial feature parameters are no longer fixed; they are freely generated by AI driven by character description (gender/age/personality/temperament) to ensure visual differentiation between characters.

### General Requirements

| Item | Constraint |
|---|---|
| Facial Features | Naturally derived from character description, no preset face shape/eye shape/eyebrow shape/nose shape/lip shape |
| Style Foundation | Chinese-style anime, neo-Chinese aesthetic, Japanese anime rendering, cel shading flat coloring, delicate brushwork |
| Temperament | Must extract overall temperament keywords from character description (e.g., elegant and gentle/refined and heroic/chivalrous and tender) and write into prompt |
| Expression | Neutral micro-expression, matching character temperament |

## VI. Base Clothing Constraints

> Base clothing has no special constraints; female wears plain colored ancient-style long dress, male wears plain colored ancient-style long robe. Formal attire is added in the costume derivative phase.

### Female Base Clothing

Plain colored ancient-style long dress, primarily in basic colors, no patterned decoration.

### Male Base Clothing

Plain colored ancient-style long robe, primarily in basic colors, no patterned decoration.

### Attire Uniform Rules

- Unified clothing style to ensure no color interference for subsequent costume overlay
- Basically covers except face/hands/neck
- Clothing style must be completely consistent across all four views
- Base clothing is merely a safe base layer, focus is on face and posture

## VII. Four-View Character Sheet Specifications

### View Definitions

| Position | View | Angle | Shot Type | Requirement | Prompt Keywords |
|---|---|---|---|---|---|
| Left 1 | Portrait Close-up | Front, eye level | Top of head to collarbone | Complete display from top of head to collarbone, face 60%+, clear facial features | portrait closeup, face detail |
| Left 2 | Front View | Front 0° | Full Body | Facing camera, arms naturally at sides, complete from top of head to feet | front view, full body |
| Right 2 | Side View | Right 90° | Full Body | Pure side profile clear, complete from top of head to feet | side view, profile, full body |
| Right 1 | Back View | Back 180° | Full Body | Back of head/back/hair ends/feet clear, complete from top of head to feet | back view, rear view, full body |

### Image Specifications

| Item | Constraint |
|---|---|
| Layout | Four views side by side left to right in one frame |
| Background | Moon white solid #E8EAF5 |
| Standing Pose | Natural stance, feet parallel slightly apart, arms naturally at sides |
| Full Body Display | Full body figure must be completely in frame from top of head to feet, no cropping |
| Close-up Display | Portrait close-up must be completely in frame from top of head to collarbone, no cropping |
| Expression | Neutral micro-expression, matching character temperament |
| Lighting | Even soft light, front key light + bilateral fill light, no hard shadows |
| Consistency | Skin tone/body type/hairstyle/face/base clothing completely consistent across four views |
| Aspect Ratio | Recommended 4:1 or 3:1 |

## IX. Constraint Rules

### Mandatory

| Number | Rule |
|---|---|
| R1 | Must be "makeup-free state" |
| R2 | Must declare base clothing (female: plain colored ancient-style long dress; male: plain colored ancient-style long robe) |
| R3 | Must declare "no hair accessories, no accessories" |
| R4 | Must specify "moon white solid background" |
| R5 | Must specify "four-view consistency" |
| R6 | Full body figures must be completely displayed from top of head to feet, no cropping |
| R7 | Must declare character height and constrain full body proportion via head-to-body ratio conversion (female default 160-170cm/6-7 heads, male default 175-185cm/6-7 heads) |
| R8 | Portrait close-up must be completely displayed from top of head to collarbone, no cropping of top of head |

### Prohibited

| Number | Prohibition |
|---|---|
| X1 | Any clothing/accessories/makeup beyond base clothing |
| X2 | Top hard light/bottom light/cool colored light |
| X3 | Excessive whitening to bloodless pallor / grayish skin |
| X4 | Complex scene backgrounds (must be solid color) |
| X5 | Exaggerated expressions/dynamic poses |
| X6 | Full body figures cropping top of head or feet, must be completely in frame from head to toe |
| X7 | Portrait close-up cropping top of head, must be completely in frame from top of head to collarbone |
| X8 | Ignoring height and head-to-body ratio constraints; height must be clearly declared and reflected in overall proportion via head-to-body ratio conversion |
