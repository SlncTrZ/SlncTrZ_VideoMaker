# Comedy & Humor · Director's Narrative Technique Skill Pack

## Introduction

This skill pack provides a comprehensive set of director's narrative technique references for the **Comedy & Humor** genre, covering the full workflow from macro-level narrative planning to micro-level storyboard execution. Applicable to any visual style.

## Core Philosophy

- **Subverted Expectation** — The essence of a laugh is "the audience thought it would go this way, but it didn't." The bigger the gap, the stronger the laugh
- **Rhythm is Comedy** — Same joke, right rhythm = hysterical, wrong rhythm = awkward. Timing is everything
- **Character Before Jokes** — The highest form of comedy isn't witty banter, but the absurdity naturally born from character personality clashes
- **Laughter Through Tears** — Pure comedy tires quickly. Plant real emotion in the absurdity, so the audience is laughing one moment and moved the next

## Usage

This skill pack serves as a narrative technique reference for the ToonFlow Director AI, automatically loaded during the story creation workflow to guide:

1. **Narrative Planning** — Determine comedy core, joke rhythm curve, scene comedy design, and sound effect timing
2. **Storyboard Generation** — Output specific shot sizes, camera movements, timing, and transition plans based on narrative planning

The two files have a clear division of labor and build on each other, forming a complete closed loop from "why shoot it this way" to "how exactly to shoot it."
