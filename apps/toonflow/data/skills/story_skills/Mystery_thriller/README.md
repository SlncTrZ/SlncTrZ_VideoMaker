# Mystery Thriller · Director's Narrative Technique Skill Pack

## Introduction

This skill pack provides a complete set of director's narrative technique references for **mystery thriller** genre stories, covering full-process guidance from macro narrative planning to micro storyboard execution. Suitable for any visual style.

## Core Concepts

- **Information Manipulation** — The essence of narrative is controlling "when the audience knows what." Suspense comes from information asymmetry
- **Unreliable Narration** — The camera itself can "lie": selective display, misleading shot scales, deceptive transitions
- **Peeling the Onion** — Each segment reveals one layer of truth while creating new questions, maintaining the audience's desire to solve the puzzle
- **Details are Clues** — Every seemingly unrelated shot could be foreshadowing. The sudden realization upon rewatching is the highest reward

## Usage

This skill pack serves as a narrative technique reference for the ToonFlow Director AI, automatically loaded during the story creation process to guide:

1. **Narrative Planning** — Determine the suspense core, information release rhythm, scene atmosphere, and sound effect direction
2. **Storyboard Generation** — Output specific shot scale, camera movement, duration, and transition plans based on narrative planning

The two files have clear division of labor and progressive layering, forming a complete closed loop from "why shoot this way" to "how to shoot it specifically."
